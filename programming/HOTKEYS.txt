MODIFIER CONVENTIONS
────────────────────

  Win + two modifiers  (Win+Ctrl+Alt)                → DisplayFusion global functions
  F-keys remapped                                    → AHK global convenience shortcuts
  CapsLock layer                                     → CapsLock macro pad

================================================================================
  DF-SNAP  —  DisplayFusion window snapping  (registry: HotKeys)
================================================================================
  Win+Ctrl+Alt+1     → Move window to primary monitor          (WindowPrimaryMonitor)
  Win+Ctrl+Alt+2     → Move window to next monitor             (MoveMonitor)
  Win+Ctrl+Alt+3     → Snap left half of current monitor       (SizeLeft)
  Win+Ctrl+Alt+4     → Snap right half of current monitor      (SizeRight)
  Win+Ctrl+Alt+5     → Snap top half of current monitor        (SizeTop)
  Win+Ctrl+Alt+6     → Snap bottom half of current monitor     (SizeBottom)
  Win+Ctrl+Alt+7     → Maximize on current monitor             (WindowCurrentMonitorMaximize)
  Win+Ctrl+Alt+8     → Move to next monitor + maximize         (MoveMonitorMaximize)

  AHK voice/snap functions built on top of these:
    SnapPrimaryLeft    = Win+Ctrl+Alt+1, then Win+Ctrl+Alt+3
    SnapPrimaryRight   = Win+Ctrl+Alt+1, then Win+Ctrl+Alt+4
    SnapPrimaryMain    = Win+Ctrl+Alt+1, then Win+Ctrl+Alt+7
    SnapSecondaryTop   = Win+Ctrl+Alt+1, Win+Ctrl+Alt+2, then Win+Ctrl+Alt+5
    SnapSecondaryBottom= Win+Ctrl+Alt+1, Win+Ctrl+Alt+2, then Win+Ctrl+Alt+6
    SnapSecondaryMain  = Win+Ctrl+Alt+1, Win+Ctrl+Alt+2, then Win+Ctrl+Alt+7
