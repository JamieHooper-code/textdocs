Spotify (Global)
----------------
  spot swallow <text>      — save current URL under key (snake_case)
  spot [spit] <text>       — open saved URL in Chrome new tab
  spot swallow <genre>     — add current URL to genre pool
  spot <genre>             — open random URL from genre pool
  Genres: ambient, folk punk, indie folk, sad boi, funk, dance,
          hip hop, Spanish, miscellaneous, local, harmony
