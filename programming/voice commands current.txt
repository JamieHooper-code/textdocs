Lockout Timer (Global)
----------------------
  lockout <n>               — plain n-min countdown, normal OSRS sound
  lockout <phrase>          — play spot phrase + muted n-min timer (default 1)
  lockout <phrase> <n>      — same with explicit minutes
  lockout <genre> [<n>]     — random genre URL + muted timer
  Accepts any phrase: hardcoded slot, hardcoded ambient track, genre name,
  or free dictation (falls back to SpotSpit with snake_case lookup).
  Spotify warm-up (3s click + 1.5s settle) is automatic; YouTube URLs get
  a 3.5s warm-up before the timer starts.
  Timer starts muted so the spot music is what you hear.

Named Clipboard Slots (swallow / spit)
--------------------------------------
  swallow <name>              — save current selection to a named slot (snake_cased)
  spit <name>                 — paste a saved named slot at the cursor
  swallow pass <name>         — save selection to a gitignored password slot
  spit pass <name>            — paste a saved password slot at the cursor
  swallow search <query>      — centered GUI listing every general + password
                                slot whose name contains all spoken keywords;
                                Slots and Passwords shown in separate sections,
                                underscores rendered as spaces, Enter/Esc closes
  Slot storage (encrypted at rest): AutoHotkey\INIDATA\CopyPasteSlots\*.enc
                                    AutoHotkey\INIDATA\CopyPasteSlotsPasswords\*.enc

Stack
-----
  stack that                 — grab entire focused field, push to stack (deletes field contents)
  pop it                     — paste top stack item into focused field
  open stack [key] [pos]     — open stack file in Notepad++ on secondary monitor
    pos: top / bot (default) / full
    e.g. "open stack", "open stack top", "open stack research", "open stack research top"
  save stack <key>           — copy current stack to named file (does not switch)
  switch stack <key>         — change active stack (e.g. "switch stack research", "switch stack main")
  dump stack                 — delete entire current stack
  list stack                 — show all stack names (current marked with *)
  Stack files: C:\Users\jamie\Desktop\Important\TEXTDOCS\programming\stacks\
