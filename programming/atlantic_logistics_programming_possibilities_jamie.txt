== LIMITATIONS ==

PHYSICAL SETUP
- I have a hand disability and use voice control + a custom macro system to code
- I'm highly functional when tasks are aligned with the abilities of my setup
- I take longer to onboard to new tools since I need to build macros for them first

REMOTE ACCESS
- RDP is not practical -- my macro setup doesn't carry over to remote desktops
- SSH / command line is likely fine, though may have a learning curve depending on the task
- Best case: work is done locally or through APIs, not by remoting into their systems

UNFAMILIAR UIs
- Tasks requiring manual operation of software I've never used will have a slower ramp-up
- There's always an initial ramp-up, but once I'm set up I move quickly

WHAT TO CLARIFY EARLY
- What does the actual workflow look like? (Files? API access? Direct DB access?)
- Will I need to remote into their systems or can I work from my own machine?
- What format is the data in?
