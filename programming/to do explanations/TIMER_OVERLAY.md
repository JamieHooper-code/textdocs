# Timer Overlay (TimerOverlay.ahk / TimerAudio.py)

Improvements queued for the timer overlay system. Headline entry in
[[Personal]] points here.

## Quotes display
Add `Scripts/quotes.txt` (one per line). Pick random at start or rotate on
tick, display under song name.

## Multiple sound folders
Either single variable at top of `TimerAudio.py`, or a weighted list of
folder paths picked per track. Candidates: OSRS, other game OSTs, lo-fi.
`Adventure.ogg` end-track could vary by active set.
